# Student Grade Management System

A console-based Java application developed to manage and analyze student grades efficiently.

## 📌 Features
- Add student details
- Store student marks using ArrayList
- Calculate average marks
- Find highest and lowest scores
- Display complete student report
- Simple and user-friendly console interface

## 🛠 Technologies Used
- Java
- Eclipse IDE
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## 📂 Project Structure

StudentGradeManagement
│
├── src
│ └── com.studentgrades
│ └── StudentGradeManager.java


## ▶️ How to Run
1. Open project in Eclipse IDE
2. Run `StudentGradeManager.java`
3. Enter student details and marks
4. View generated report and statistics
